== Description ==

实现死链接生成集中链接，并提交到百度、360、头条搜索、谷歌google、Bing、神马等搜索引擎。如果网站存在大量死链，将影响网站的站点评级，建议存在死链内容的网站，使用本工具。

## 插件功能
* 支持手动添加一条死链接URL
* 支持生成死链接badurl.txt
* 支持集合死链接提交到百度、死链提交到360、死链提交到头条搜索、死链提交到google、死链提交到Bing、死链提交到神马等搜索引擎
* 支持自动扫描所有文章不存在或已删除死链接，自动生成文件，非常方便站长用户使用+
